# w281: Computer Vision (Summer 2025)

Welcome to the repository for **w281: Computer Vision** at Berkeley, Summer 2025.

## Course Overview

This course introduces the fundamentals of computer vision, covering key concepts, algorithms, and practical applications. Students will learn how computers interpret and process visual information from the world.

## Topics Covered

- Image processing and filtering
- Feature detection and matching
- Object recognition and classification
- Deep learning for vision (CNNs)
- Image segmentation
- 3D vision and geometry
- Applications: face recognition, scene understanding, and more

## Prerequisites

- Python programming experience
- Basic knowledge of linear algebra, calculus, and probability
- Familiarity with machine learning concepts is helpful
